on making podcasts in April

I wanted to do something a little bit out of my comfort zone, so I figured I’d try a daily podcast for a month. Which turns out to be quite a ballache to get started. But I’ve got it up and running.
This is my setup

  • Server
    1. Sinatra app running on Heroku
    2. MP3s hosted on Amazon S3
  • Clients
    1. Garageband
    2. Skype
  • Distributors
    1. Feedburner
    2. Mailchimp

The code, hosted on Github ( of course ) is a small webapp that contains a few tests to ensure the data you put in to the server is valid before sending it off to heroku for hostage.

From there Feedburner eventually picks up on the changes and updates my RSS feeds, once that’s updated Mailchimp picks up any changes every 8pm and sends out emails to people that prefer to get a mail update. Right now they’re both about as popular as each other.
