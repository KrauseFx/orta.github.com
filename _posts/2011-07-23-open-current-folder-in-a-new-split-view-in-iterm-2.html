iTerm insta-tab split vertical.

Want to load up a local copy of you application, but dont want it hogging the terminal, nor is sending it into the background hidden away making using git and reading logs a chore, I get around this by creating a split view that lets you quickly open a new tab, then you can run your server command and go back to your other view. 

tab(){
osascript -e "
tell application \"System Events\"
	tell process \"iTerm\"
		click menu item \"Split Vertically\" of menu 1 of menu bar item \"Shell\" of menu bar 1
	end tell
end tell

tell application \"iTerm\"
 tell the current terminal
  tell the current session
   write text \"cd $(pwd)\"
  end tell
 end tell
end tell"
}
